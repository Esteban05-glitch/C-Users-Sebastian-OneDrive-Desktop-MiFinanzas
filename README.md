# Gestor de Finanzas Personales

Programa en Python para ingresar tus gastos diarios, calcular el total gastado, mostrar cuánto te queda de presupuesto y generar alertas si estás cerca de excederte.

## Uso

1. Ejecuta el archivo `gestor_finanzas.py`.
2. Ingresa tu presupuesto total.
3. Agrega los gastos uno por uno.
4. Consulta el resumen cuando quieras.

### Ejecución

```bash
python gestor_finanzas.py
```

## Características

- Registro de gastos diarios
- Cálculo del total gastado
- Cálculo del presupuesto restante
- Alertas cuando el gasto alcanza el 75%, 90% o excede el 100% del presupuesto
